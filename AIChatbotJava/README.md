# AI ChatBot using Java

## Overview

This project is a simple AI ChatBot developed using Java. The chatbot interacts with users through a command-line interface and provides rule-based responses to frequently asked questions.

## Features

* Interactive communication
* Rule-based response system
* Basic Natural Language Processing (NLP) using text matching
* Frequently Asked Questions (FAQ) support
* Real-time user interaction
* Exit command support

## Technologies Used

* Java
* Scanner Class
* Conditional Statements
* NLP Techniques (Keyword Matching)

## Sample Questions

* hello
* java
* python
* internship
* bye

## How to Run

Compile the program:

javac ChatBot.java

Run the program:

java ChatBot

## Output

The chatbot responds to user queries and provides predefined answers based on keywords.
